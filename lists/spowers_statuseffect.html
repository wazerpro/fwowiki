Status Effect Powers

This is a selection of the powers that inflict a status effect to the target. Each status effect has its own disabling effect, which can be extremely useful (if you're using it) or extremely troublesome (if you're inflicted with it). The rank of a power that inflicts a status effect determines the success rate as well as the duration (and in the case of an aoe, sometimes the range).
